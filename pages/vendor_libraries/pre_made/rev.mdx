## Rev Robotics Lib

Arfur, an older rust project has already created cxx bind tooling for rev lib it can be imported with

```toml copy
[dependancies]
arfur-rev = "0.0.1"
```

If it is out of date, the tooling to create a modern version is available in the main [Arfur repo](https://github.com/arfur-rs)
with instructions.
