## Hello World

In order to confirm you have everything correctly setup for your development machine, a simple hello world program in
rust serves to test rustup, rustc, cargo, & your IDE are correctly configured

import { Steps } from "nextra/components";

<Steps>
    ### Create Project
    open a terminal and run `cargo new hello_world`

    ### Open
    open the created directory in your IDE

    ### Check
    check that syntax highlighting is functioning in src/main.rs

    ### Run
    run `cargo build` in the directory

    ### Results
    If your terminal prints the message below, you're ready to go with rust.

</Steps>

```
./targetdebug/hello_world
Hello, world!
```

### Help

If this process failed, resources are available at [Rust Lang Docs](https://www.rust-lang.org/learn/get-started) and
direct help is available in our discord server, just click the icon in the top right to join.
