## VS Code

<Steps>
    ### Download & Install
    You can find the downloads and install instructions on the [VS Code website](https://code.visualstudio.com/). It is available for all major operating systems and is completley free.

    ### Install Extensions

    In the extensions tab, you can either search for and install the Rust Extension Pack, or install the following extensions individually:
    - rust-analyzer
    - Even Better TOML

    ### Final Steps

    - Restart VS Code after installing the extensions.
    - Run `rustup --update` to make sure you have the latest version of rust installed.
    - Configure version control (git or github desktop) if you haven't already.

</Steps>

## CLion

<Steps>
    ### Download
    - Download [JetBrains CLion](https://www.jetbrains.com/clion/). If you are a student, you can get a free license through the [JetBrains Student Program](https://www.jetbrains.com/community/education/#students).

    ### Plugins
    - Install the official Rust plugin
    - Go to settings / language & frameworks / rust / external linters and select clippy.
    - If your machine has reasonable compute power enable "Run external linter to analyze code on the fly"

    ### Final Steps

    - Run `rustup --update` to make sure you have the latest version of rust installed.
    - Configure version control (git or github desktop) if you haven't already.

</Steps>
