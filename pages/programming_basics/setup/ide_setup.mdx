import { Steps } from "nextra/components";

## VS Code

<Steps>
    ### Download
    - Download [VS Code](https://code.visualstudio.com/)

    ### Configure Extensions

    - Install the rust-analyzer from the extension tab
    - Install the `.toml` file extension

    ### Version Control

    - Configure git

    ### Final Check

    - run `rustup --update`

</Steps>

## CLion

<Steps>
    ### Download
    - Download [JetBrains CLion](https://www.jetbrains.com/clion/)

    ### Plugins
    - Install the official Rust plugin
    - Go to settings / language & frameworks / rust / external linters and select clippy, if your machine has reasonable
    compute power enable "Run external linter to analyze code on the fly"

    ### Version Control
    - Configure git

    ### Final Check
    - run `rustup --update`

</Steps>
